# UFCWebPageHTML
UFCPageWithHTML is a web project developed using HTML that presents information about some of the best and most influential fighters in the history of the UFC. The website includes biographies, fighting styles, martial arts disciplines, weight divisions, and notable achievements of several fighters.
The main goal of this project is to practice and improve my front-end development skills using HTML while creating structured web pages that display clear and informative content about mixed martial arts (MMA) and its most iconic athletes.
